# SCOPE — AI生成应用场景方法论

> **S**trengthen（增强目标）→ **C**onstrain（约束预警）→ **O**riginate（创意生成）→ **P**ath（路径规划）→ **E**ffect（效应匹配）

---

## 项目概述

SCOPE 是一套系统性地生成"AI能用在哪儿、怎么用"的方法论。它借鉴 TRIZ（发明问题解决理论）的思想结构，但完全独立，专注于**AI赋能业务**这一垂直领域。

核心命题：**把寻找AI应用场景，从"拍脑袋"的艺术，变成"按图索骥"的科学。**

## 方法论架构（五层）

### 第一层：业务场景参数（6改善 × 6恶化）

| 改善参数（想提升什么） | 恶化参数（担心什么） |
|----------------------|-------------------|
| 运营效率 Operational Efficiency | 实施复杂度 Implementation Complexity |
| 客户满意度 Customer Satisfaction | 可解释性 Explainability |
| 创新能力 Innovation Capability | 对人的依赖度 Human Dependency Risk |
| 风险控制 Risk Control | 成本 Cost |
| 收入增长 Revenue Growth | 数据隐私风险 Data Privacy Risk |
| 决策速度 Decision Speed | 系统安全性 System Security |

### 第二层：36个AI业务发明原理（六大类）

**重构流程与体验**: 智能分割 / 认知自动化 / 极致个性化 / 动态定价与匹配 / 无缝交互 / 流程重塑

**增强决策与洞察**: 预测与推演 / 智能溯源 / 智能编排与调度 / 风险管理师 / 情境感知 / 群体智慧涌现

**重塑生产关系**: 人机环协同 / 数字员工·智能体 / 人机共创 / 能力即服务 / 双边网络桥接 / 智能合约

**颠覆产品与资产**: 数字孪生 / 软件定义X / 服务机器人化 / 生成式设计 / 超个性化生产 / 知识产权生成

**转变行为与赋能**: 智能教练与行为助推 / 能力延伸

**范式跃迁**: 变废为宝 / 跨越中界 / 自我进化系统 / 生成式创造力 / 复合感知与响应 / 预见性创新 / 蜂群智能 / 自适应安全 / 客户共创生态 / 元能力

### 第三层：矛盾矩阵（6×6，210案例统计填充，填满率100%）

横轴=恶化参数，纵轴=改善参数。交叉格中填入最相关的AI发明原理（带信度等级）。V3.2最后一格S5×C3已填补。

### 第四层：AI业务进化路线（7条）

1. **洞察深度**: 事后描述 → 实时洞察 → 预测性警报 → 规范建议 → 自主行动
2. **决策自主度**: AI辅助人 → AI增强人 → AI替代人(特定) → AI自主执行 → AI完全自主
3. **产品形态**: 流程自动化 → 智能产品化 → 生态平台化
4. **技术范式**: 基于规则 → 基于模型 → 基于生成 → 基于自进化
5. **交互方式**: 菜单/表单 → 自然语言 → 多模态融合 → 无感交互
6. **Agent架构**: 单Agent → 多Agent协同 → Agent生态
7. **价值分配**: 层级中介 → 平台中介 → AI使能的去中介化

> ⚠️ 所有路线均含"回滚"分支——当恶化参数失控时可能触发。

### 第五层：AI能力效应库

| 业务需要 | 推荐AI技术 |
|---------|-----------|
| 理解/生成人类语言 | LLM (GPT/Claude/DeepSeek/Qwen) |
| 识别图像/视频中的对象 | 计算机视觉 (CNN/ViT/YOLO) |
| 从文档提取结构化信息 | OCR + 信息抽取 + LLM |
| 预测设备故障/需求 | 时序预测 (LSTM/Transformer/GBDT) |
| 发现异常/欺诈 | 孤立森林/自编码器/GNN |
| 个性化推荐 | 协同过滤/上下文Bandit/LLM |
| 知识图谱推理 | 图嵌入/路径排序/LLM+KG |
| 动态调度优化 | 强化学习/运筹优化/约束求解 |
| ... | *(共14个核心映射)* |

## 使用流程（SCOPE五步法）

```
1. Strengthen  明确要提升的核心业务目标（查6个改善参数）
       ↓
2. Constrain   识别可能恶化的业务约束（查6个恶化参数）
       ↓
3. Originate   查矛盾矩阵 → 获得AI发明原理推荐（带信度等级）
       ↓
4. Path        评估当前在7条进化路线上的位置 → 规划三级跳路径
       ↓
5. Effect      查AI能力效应库 → 找到实现原理所需的具体技术
```
