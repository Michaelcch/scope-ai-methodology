FIG_W = 24   # 英寸

**Step 5: 格内文字格式化**
```python
def format_cell_text(cell):
    """返回 (lines, colors, sizes, weights) 四个等长列表"""
    lines = []
    colors = []
    sizes = []
    weights = []

    conf = cell['confidence']
    sym = {'high': '■', 'medium_high': '□', 'medium': '△', 'low': '▽'}[conf]
    text_color = 'white' if conf in ('high', 'low') else '#212121'

    # 第1行：信度符号 + N
    lines.append(f'{sym} N={cell["N"]}')
    colors.append(text_color)
    sizes.append(10)
    weights.append('bold')

    # 中间行：差异化原理（自动换行，每行最多约26字符宽）
    if cell['principles']:
        p_str = ', '.join(str(p) for p in cell['principles'])
        wrapped = wrap_text(p_str, max_chars=26)
        for w in wrapped:
            lines.append(w)
            colors.append(text_color)
            sizes.append(7.5)
            weights.append('normal')

    # 末行：基础原理
    lines.append(f'+🔧2, 🔧7, 🔧14')
    colors.append('#78909C')
    sizes.append(6.5)
    weights.append('normal')

    return lines, colors, sizes, weights
```

## 六、注意事项

1. **不放禁忌矩阵**：本海报仅展示矛盾矩阵 + 原理索引，禁忌体系（V5.2第六节）不纳入画面。
2. **S7/C7 空数据**：V5.2中S7和C7为扩展维度，数据积累中。对应格子用灰色标示，不加原理推荐。
3. **S5-C2 ⚠️标记**：P4(动态定价)在C2恶化下须加 `(慎)` 标记，提醒使用者注意公平性。
4. **C3子关切**：C3（对人的依赖度）包含 C3a技能退化 / C3b就业替代 / C3c情感依恋，可在C3列头加脚注标记 `*`。
5. **P4含失败案例**：原理索引中P4应加 `⚠️` 标记。
6. **色盲友好**：绿色 `#1B7837`、金色 `#E6C800`、橙色 `#E68A00`、红色 `#D73027` 四色在红绿色盲下仍可区分亮度差异。
7. **格内原理过多时的处理**：S3-C1(13个差异化原理)为最密格，需确保用更小字号(6.5pt) + 分行显示，不能溢出格边界。
8. **留白原则**：矩阵区与原理索引区之间保留至少0.5英寸间隙，文字不重叠。
